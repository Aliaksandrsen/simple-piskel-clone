# Simple piskel clone
- ES 2015 features
- Canvas
- Google Sign-In
- Webpack
- Babel
- ESlint (config-airbnb-base)
- EditorConfig
- Jest testing

## Demo 
https://piskel.netlify.com/

## Instruction

use **"npm run build"** for compile the project
   
use **"npm run start"** for run devserver

use  **"lint"** for lint

use **"npm test"** for test
